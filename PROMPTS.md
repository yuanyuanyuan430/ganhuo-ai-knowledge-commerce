# Prompts for ganhuo-ai-knowledge-commerce

这些 prompt 可以直接复制到 Codex / 干活 AI 中使用，把一个知识付费想法变成可执行的内容、信任、交付和复盘系统。

## 1. 知识付费流水线设计

```text
使用 ganhuo-ai-knowledge-commerce，帮我设计一个 AI 辅助知识付费流水线。

细分品类：
目标用户：
已有资料：
已有个人经验：
计划渠道：
计划产品：
价格范围：

请输出：
1. 产品定位；
2. 来源卡和知识库结构；
3. 公众号内容模板；
4. 知识星球周交付节奏；
5. 私域跟进路径；
6. 自动化分工；
7. 30/60/90 天执行表；
8. 每周复盘指标。
```

## 2. 公众号到知识星球转化路径

```text
使用 ganhuo-ai-knowledge-commerce，帮我把公众号内容设计成能承接知识星球转化的路径。

公众号主题：
目标读者：
知识星球交付内容：
已有文章：
想卖的入门产品：

请输出 10 个选题、每篇文章的信任点、CTA、私域承接话术、星球周交付主题和复盘指标。
```

## 3. 资料整理成知识库

```text
使用 ganhuo-ai-knowledge-commerce，帮我把这些资料整理成可复用知识库。

资料列表：
目标产品：
目标用户：
已有案例：

请输出来源卡字段、主题簇、FAQ、案例库、模板库、术语表、反对意见库，以及后续写公众号和星球内容的调用方式。
```

## 4. 知识星球 30 天交付计划

```text
使用 ganhuo-ai-knowledge-commerce，帮我设计一个知识星球 30 天交付计划。

```

使用 ganhuo-ai-knowledge-commerce，帮我把下面这段销售表达改得更可信、更有转化力。
